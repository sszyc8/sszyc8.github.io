介绍

  • 使用Go-gin开发的md解析博客。
  • 沉浸式写作，Git读取部署。

规范说明

  • 文件名：格式为日期-名称，例如2020-02-02-第一篇笔记
  • 资源名：统一放在res下，子目录名称与所引用的md文件名一致，例如res/2020-02-02-第一篇笔记/xx.png
  • 文章头：
  1. 第一行文章发表状态，否/不发表、是/发表。
  2. 第二行文章ID，后期通过这个ID会实现评论以及其他工呢。
  3. 第三行文章标题。
  4. 第四行文章分类。
  5. 第五行文章标签，通过|来区分多个标签。
  6. 第六行创建时间。
  7. 第七行文章的简介。

使用说明

  • 配置app/conf/basic.yml
